Descripcion del proyecto: Aplicación de consola en Java para gestionar actividades de usuarios. Permite registrar, modificar y eliminar actividades almacenadas en archivos, además de generar análisis como la actividad más realizada y estadísticas por usuario.
 
Estructura: 
tallerPOO.java = Clase principal

Usuarios.txt y Registros.txt = Archivos de texto 

Instrucciones de ejecución:
 1. Abrir el proyecto en un compilador de codigo 
 2. Verificar que los archivos de texto se encuentren dentro del proyecto, en caso contrario al inicio del codigo se encuentran los String con las rutas de los respectivos archivos, pueden reemplazarse por las rutas personalizadas
 3. Ejecutar la clase principal.
